# Ant Design Demos

这是一个使用 React、webpack 和 Ant Design 搭建的前端项目。

## 安装依赖

```bash
npm install
```

## 开发模式运行

```bash
npm start
```

## 构建生产版本

```bash
npm run build
```

## 项目结构

- `public/` - 静态资源目录
- `src/` - 源代码目录
  - `App.js` - 主应用组件
  - `App.css` - 应用样式
  - `index.js` - 应用入口文件
- `webpack.config.js` - webpack 配置文件
- `package.json` - 项目配置和依赖

## 技术栈

- React 18
- Webpack 5
- Ant Design 5
- Babel
